Open your JavaScript console and type chat('Your Name'); to start the chat.

Then use chat.join('Room Name'); and chat.leave('Room Name'); to join/leave a chat room. You can be member of multiple rooms at the same time.

After having joined a room use chat.room('Room Name').send('Your Message'); to send a message in a room.

If you want to buidl an interface for this chat you can fork me on GitHub.
